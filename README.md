# legend

A stateless, composable React UI primitive for surfacing the most relevant
keyboard hint for the current app state. Ships alongside a minimal log
explorer that demonstrates it in use.

In active development.

## Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- CSS Modules with a global design-token foundation
- Vitest + React Testing Library
- pnpm

## Getting started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.
